# 🎮 [Tool] RPGM-Translator v1.0.0 — Translate RPG Maker MV/MZ games automatically!

Hey everyone! 👋

It's a simple GUI tool that extracts all the translatable text from an RPG Maker MV/MZ game, translates it with your favorite backend, and patches the game files in-place. You can also export a clean patch to share with others. No manual JSON editing, no headaches. 😎

---

## 📋 Quick Info

- **Name:** RPGM-Translator

---

## ✨ What It Can Do

- 🔍 Auto-detects **RPG Maker MV/MZ** games.
- 📝 Extracts translatable strings from:
  - `Map*.json` — dialogues, choices, scrolling text
  - `CommonEvents.json` — including script literals / Oracle tips
  - `System.json` — game title, terms, labels
  - `Items.json`, `Weapons.json`, `Armors.json`, `Skills.json`, `States.json`, `Enemies.json`, `Actors.json`, `Classes.json`
  - `js/plugins.js` — plugin parameters
- 🌍 Multiple translation backends: **Google Turbo**, **Bing Ultra**, **OpenRouter**, **Local Llama**.
- 🔎 Editable translation table with filters (All / Translated / Untranslated), file filter, and live search.
- 🔄 **Replace All** — find and replace specific words across all translations or filtered items only.
- 🧹 **Clear cache** button to wipe global and local caches.
- 💾 In-place patching with automatic `www/data` backup.
- 📦 **Export patch** to distribute translated `www/data`.
- 🛠️ One-click **Forge cheat mod** install for MV/MZ.
- 🔄 **Restore Backup** to revert to original game data.
- 🗂️ **RPG Maker dialogue prefix handling** — general logic to handle RPG Maker dialogue prefixes with special separators (`<`, `>`, `}`, `{`, `^`) while translating only the dialogue content.
- 🧠 **Smart $gameVariables.setValue handling** — intelligently distinguishes between short NPC names (preserved) and dialogue phrases (translated) in script literals.
- 👤 **Automatic character name extraction** — automatically extracts character names from `Actors.json` to preserve them from translation.
- 🛡️ **Dynamic filename pattern exclusion** — excludes dynamic filename patterns (e.g., `CHR-`, `-Body`, `-Face`) from translation to prevent crashes.

## 🔄 How to Use

1. 🎮 **Select the game** folder or `.app` (macOS).
2. 🧠 Click **Analyze & Translate**.
3. ✏️ Review or edit any string in the table.
4. 💾 Click **Save** to apply the translation — a backup is created automatically!
5. 📦 Click **Export** to create a distributable patch.
6. 🛠️ Optional: click **Install Forge Cheat Mod** to add the overlay.

## 🆕 Changelog

### v1.0.0
- 🎯 **Delimiter-based escape code handling** — preserves exact positioning of escape codes (colors, icons) by treating them as delimiters and translating only text segments between them.
- ⚡ **Optimized segment translation** — batch processes all text segments for improved performance.
- 🔧 **Enhanced RPG Maker prefix recognition** — supports multiple consecutive special characters in prefixes (e.g., `<<`, `>>`).
- 👤 **Character name preservation in dialogues** — replaces character names with placeholders before translation and restores them after, preventing unwanted name translations.
- 🔍 **File filter** — filter strings by source file (e.g., Items.json, Armor.json) for easier navigation.
- 🔄 **Replace All** — find and replace specific words across all translations or filtered items only, with case-sensitive option.
- 🐛 **Short text translation fix** — workaround for very short text not being translated by Google due to Unicode special characters.
- 🧹 Added **Clear cache** button.
- 🔤 Added extraction/translation of string literals inside script commands (e.g., Oracle tips).
- 🗂️ **RPG Maker dialogue prefix handling** — general logic to handle RPG Maker dialogue prefixes with special separators (`<`, `>`, `}`, `{`, `^`) while translating only the dialogue content.
- 🧠 **Smart $gameVariables.setValue handling** — intelligently distinguishes between short NPC names (preserved) and dialogue phrases (translated) in script literals.
- 🛡️ **Dynamic filename pattern exclusion** — excludes dynamic filename patterns (e.g., `CHR-`, `-Body`, `-Face`) from translation to prevent crashes.
- 🎉 Initial stable release with Google/Bing/OpenRouter/Llama backends, editable table, and Forge cheat mod integration.

## 📝 Notes

- A single backup (`www/data_bak_original`) is created on first save and is never overwritten. Use **Restore Backup** to revert to original game data.
- Use **Clear cache** before re-analyzing a game to start fresh.
- `forge.js` is bundled in the release; replace it and rebuild if you need a newer Forge version.

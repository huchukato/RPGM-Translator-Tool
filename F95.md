## вЬ® What It Can Do

- рЯФН Auto-detects **RPG Maker MV/MZ** games.
- рЯУЭ Extracts translatable strings from:
  - `Map*.json` вАФ dialogues, choices, scrolling text
  - `CommonEvents.json` вАФ including script literals / Oracle tips
  - `System.json` вАФ game title, terms, labels
  - `Items.json`, `Weapons.json`, `Armors.json`, `Skills.json`, `States.json`, `Enemies.json`, `Actors.json`, `Classes.json`
  - `js/plugins.js` вАФ plugin parameters
- рЯМН Multiple translation backends: **Google Turbo**, **Bing Ultra**, **OpenRouter**, **Local Llama**.
- рЯФО Editable translation table with filters (All / Translated / Untranslated) and live search.
- рЯІє **Clear cache** button to wipe global and local caches.
- рЯТЊ In-place patching with automatic `www/data` backup.
- рЯУ¶ **Export patch** to distribute translated `www/data`.
- рЯЫ†пЄП One-click **Forge cheat mod** install for MV/MZ.
- рЯФД **Restore Backup** to revert to original game data.
- рЯЧВпЄП **RPG Maker dialogue prefix handling** вАФ general logic to handle RPG Maker dialogue prefixes with special separators (`<`, `>`, `}`, `{`, `^`) while translating only the dialogue content.
- рЯІ† **Smart $gameVariables.setValue handling** вАФ intelligently distinguishes between short NPC names (preserved) and dialogue phrases (translated) in script literals.
- пњљ **Automatic character name extraction** вАФ automatically extracts character names from `Actors.json` to preserve them from translation.
- рЯЫ°пЄП **Dynamic filename pattern exclusion** вАФ excludes dynamic filename patterns (e.g., `CHR-`, `-Body`, `-Face`) from translation to prevent crashes.
